

# EduClash
EduClash is a web app that takes lecture slides and turns them into interactive quizzes using AI.
You can add friends, create teams and compete with each other to enchance revision.


## What is the problem?

Most students struggle to stay engaged
with lecture slides.
This is exasperated by some lecturers
making the slides so dry, you start seeing an
oasis in them.
Revision starts feeling passive and
repetitive, leading to poor retention and
low motivation.

## How we address it?

By turning your lecturer’s boring slides into a
dynamic, mentally stimulating dopamine
machine. Earn XP, keep streaks and climb the leaderboards in your education journey.

## Key features:
>Turn lecture slides into quizzes
>
>Answer multiple choice and true/false questions
>
>Invite your friends to compete
>
>Compare high scores and win/loss ratios
>
>Keep track of daily progress
>
>LEARN!

## Tech Stack

> Frontend - React, Tailwind CSS, React Router

> Backend/ API  - Supabase, PostGres, Deno Edge Function

> Database - Supabase (DB + Auth)

> AI - Gemini API
